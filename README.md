# Nutrio – AI-Powered Nutrition Tracking App

Nutrio is an Android MVP designed to help users track nutrition more easily through AI food recognition, barcode-based nutrition lookup, and personalized health dashboards.

## Overview

Manual calorie tracking can be time-consuming and difficult to maintain consistently. Nutrio aims to reduce this friction by allowing users to identify food, retrieve nutrition data, and monitor their daily intake in a simple mobile experience.

## Key Features

- AI-based food recognition
- Barcode nutrition lookup using Edamam API
- Daily food logging
- Calories and nutrition dashboard
- BMI and TDEE calculation
- Firebase authentication and user data management
- Mobile-first user interface

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Edamam API
- Android Studio

## Product Problem

Many users want to track their nutrition but find manual food logging inconvenient. They may forget to log meals, struggle to estimate calories, or feel overwhelmed by complex fitness apps.

## Solution

Nutrio simplifies nutrition tracking by combining food recognition, barcode lookup, and a clear dashboard. The goal is to help users understand their eating habits with less manual effort.
